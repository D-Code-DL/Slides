What is a user story?

"A user story is a tool used in Agile software development to capture a description of a software feature from an end-user perspective." (searchsoftwarequality.techtarget.com/definition/user-story)

What is the usual format of a user story?

As a (type of) user, I want to ____(some goal)___ so that ___(some reason)__.

The type of user doesn't have to be defined, but it can for more detail.

What really stood out to you about your visit to Detroit Labs?

What (if anything) did any of the guest speakers say that surprised you?

What new thing did you learn about the tech industry through the visit?

Final Webpage Project (Project #3)

Let's review the paper we are handing out.

The purpose of this website project is to create something that others can enjoy and that will bring of the skills you have gained as web developers together. You will be making a text adventure game!

A text adventure game is an interactive story that allows the user to select between choices that changes the course of the story.

This game should be fun, but it should also meet the sandards on the piece of paper that we will be handing out.

It is an individual project.

MVP (minimum viable product):

  • Use CSS
  • Use Javascript
  • Has header with story title
  • Single HTML page (text updates)
  • Has 2+ for user selection
  • Has 2+ level deep story tree

2.0

  • Use color scheme (elements and background)
  • Use w3 method for setting event listeners
  • Has 1+ images
  • Requests username and prints on page
  • Use external font
  • Text of story is centered on page with borders around it
  • Has subtitle that updates as user progresses through the story

3.0

  • Create intro page that accepts 2+ types of user info (e.g. name and age) using form elements
  • Use event listener for onmouseover to change the color of an element
  • Use sound effects
  • Incorporate user information into story
  • Has 2+ images that change based on user location in story
  • Has progress bar that updates as user progresses through the story
  • Has 1+ animation
  • Provide user ability to start over at the end of the story

Plan Ahead
